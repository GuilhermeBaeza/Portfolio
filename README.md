## ✨ Funcionalidades

O layout foi pensado para ser dinâmico e interativo, sem a necessidade de frameworks pesados no momento:
- **Dark/Light Mode:** Alternância de temas perfeitamente integrada com variáveis CSS.
- **Partículas Dinâmicas (Canvas):** Fundo interativo em JavaScript puro que reage ao movimento do mouse.
- **Cursor Customizado:** Efeito de "blend mode" no cursor durante a navegação no tema escuro.
- **Responsividade Total:** Adaptado para funcionar perfeitamente em telas de desktop, tablets e smartphones (Media Queries).
- **Integração de E-mail:** Formulário de contato que direciona automaticamente a mensagem estruturada para os clientes de e-mail (Gmail/Outlook).

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído "Vanilla" para consolidar os fundamentos da web:
* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização com variáveis (Custom Properties), CSS Grid, Flexbox e animações `@keyframes`.
* **JavaScript (ES6+):** Manipulação do DOM, Intersection Observer para animações de *scroll*, Canvas API e lógica do tema.
* **[Lucide Icons](https://lucide.dev/):** Biblioteca de ícones renderizados via SVG de forma leve.
* **Vercel:** Plataforma escolhida para o deploy e hospedagem contínua.
